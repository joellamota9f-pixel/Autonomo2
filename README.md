# Proyecto Final – Desarrollo de Software

## Descripción
Este proyecto corresponde al final, cuyo propósito es analizar y desarrollar una aplicación de software aplicando los conocimientos adquiridos durante las semanas de clase. El proyecto evidencia el proceso de desarrollo desde la configuración del entorno hasta la implementación de funcionalidades mediante estructuras lógicas, repetitivas y estructuras de datos.

---

## Objetivo del Proyecto
Desarrollar una aplicación funcional aplicando los conocimientos adquiridos en la asignatura, con el fin de analizar y comprender la arquitectura y las funcionalidades de un sistema de software. El proyecto busca poner en práctica el uso de estructuras lógicas, estructuras repetitivas y buenas prácticas de programación, así como el uso de herramientas de control de versiones mediante GitHub para la gestión del desarrollo.

---

## Alcance del Proyecto
El proyecto abarca desde la configuración inicial del entorno de desarrollo hasta la implementación de las funcionalidades principales del sistema. Incluye la creación de diagramas de flujo para representar la lógica del programa, la programación de condicionales, ciclos repetitivos y estructuras de datos.

El sistema desarrollado permite demostrar el funcionamiento correcto de las funcionalidades implementadas; sin embargo, no contempla la implementación de bases de datos, interfaces gráficas avanzadas ni despliegue en producción, ya que el enfoque principal es académico y orientado al aprendizaje de los fundamentos del desarrollo de software.

---

## Versiones del Proyecto

### Versión 1.0 – Trabajo Autónomo 2
Corresponde a la primera versión del proyecto, donde se realizó la configuración del repositorio en GitHub, la elaboración de los diagramas de flujo y el desarrollo del avance inicial de la codificación. En esta versión se implementaron estructuras lógicas y repetitivas básicas, permitiendo la interacción inicial con el usuario y el funcionamiento general del sistema.

### Versión 2.0 – Versión Final
En la versión final del proyecto se amplían y optimizan las funcionalidades del sistema, incorporando el uso de **estructuras de datos**, tales como listas, diccionarios u otras según la necesidad del programa. Estas estructuras permiten una mejor organización, almacenamiento y manipulación de la información, mejorando la eficiencia y escalabilidad del software desarrollado.
